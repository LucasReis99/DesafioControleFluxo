O desafio consiste em criar um programa em Java que recebe dois números inteiros do usuário via terminal e conta progressivamente a partir do primeiro número até o segundo número, imprimindo cada número no console. Se o segundo número fornecido for menor ou igual ao primeiro, o programa lança uma exceção personalizada chamada ParametrosInvalidosException.



O programa é implementado em duas classes: Contador e ParametrosInvalidosException. A classe Contador contém o método main onde são solicitados os números e a lógica de contagem é invocada. A contagem é realizada no método contar, que verifica se o segundo número é maior que o primeiro antes de iniciar a contagem. Se não for, lança a exceção ParametrosInvalidosException.
